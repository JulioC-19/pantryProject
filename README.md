## Troubleshooting
### Enviroment Issues
#### Android Rebuild
- Uninstall application from emulator
- Close out emulator windows 
- Delete android build folder under app folder: android\app\build
- ` rmdir node_modules `
- ` npm cache clean --force `
- ` npm install `
- ` npm run start `
- ` npm run android `
